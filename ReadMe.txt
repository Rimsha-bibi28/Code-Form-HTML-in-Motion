 HTML Essentials: Tables • Lists • Forms

A minimal, accessible website built to practice **tables**, **lists**, and **forms** from Afshan's class.


- Semantic, accessible **table** with caption, thead/tbody/tfoot
- **Lists**: ordered, unordered, and definition list (glossary)
- **Form** with labels, constraints, and client-side validation
- Clean, responsive styling (no frameworks)

 Getting Started
1. **Download or clone** this repo
2. Open `index.html` in your browser

> No build tools required. Pure HTML/CSS/JS.

 Deploy to GitHub Pages
1. Push the repo to GitHub
2. Go to **Settings → Pages**
3. Set **Source** to `Deploy from a branch`, pick `main` and `/ (root)`
4. Save — your site will be live at `https://<username>.github.io/afshan-task-site/`
